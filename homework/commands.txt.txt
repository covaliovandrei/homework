git config

Эта команда устанавливает имя автора и адрес электронной почты соответственно для использования с вашими коммитами.

git init

Эта команда используется для запуска нового репозитория.

git clone

Эта команда используется для получения репозитория по существующему URL-адресу.

git add

Эта команда добавляет файл в промежуточную область.

git commit

Эта команда постоянно записывает или делает моментальные снимки файла в истории версий.

git diff

Эта команда показывает различия файлов, которые еще не подготовлены.

git reset

Эта команда удаляет файл из стадии, но сохраняет содержимое файла.

git status

Эта команда выводит список всех файлов, которые должны быть зафиксированы.

git rm

Эта команда удаляет файл из вашего рабочего каталога и поэтапно удаляет его.

git log

Эта команда используется для просмотра истории версий для текущей ветки.
